title: "Ressl provider snapshots"
description: "Spin up hosted mock SaaS APIs (Salesforce and more) with a public HTTPS URL, seed data, and a TTL."
---

Ressl lets you create **provider snapshots** — short-lived, hosted copies of a mock SaaS API your organization has access to.

Each snapshot gets a public base URL such as:

```text
https://{snapshotId}.salesforce.mock.ressl.cc
```

Call that host like the real provider. When the TTL expires, the snapshot is torn down.
